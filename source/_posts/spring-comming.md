## 2026年3月28日 20点12分

{% quote blue %}
春天、来了
{% endquote %}

今天学校有春招双选会，来的企业其实不算多，够得上“中大厂”的也就那几家，其他很多的都是地方公司企业。

我觉得可能只有我这种人才会想出特地把压箱底的白衬衫从衣柜里翻出来，配上黑色裤子，好久不捯饬自己衣装的人假扮成求职者的样子，这种坐牢的损招。春天是真的要来了，午后二十度的气温，热的我把宣传册当小扇子扇。

衬衣有些皱了，其实我一直挂在衣柜里，只是不穿，总会有些痕迹，加上之前搬宿舍估计随便塞在了行李箱的某个角落。但我只有这一件白色衬衣，上一次穿也许得追溯到三年前。那时候我高三，在同样某个阳光明媚的周六下午，学校举办了18岁成人仪式。

当时是我第一次穿学校的正装校服，我还记得当时从家出门时心中的忐忑，坐着我爸的车去学校都感觉自己变成了商业人士。从车上下来的时候应该有人给我打伞——欸，没有啊，好遗憾。

藏青色西装，衣领有些硬的白衬衣，特意准备的小皮鞋，妈妈送给我的漂亮发卡，还有在生物课上突然被我搞坏、最后只能用夹子夹在衣领上的领花。阳光从参天的梧桐树缝隙中落下，变成一地的碎金，爬山虎绿得滴水，风轻轻的。我在走廊上遇到了喜欢的男生，和他开玩笑说自己看起来像是卖保险的。

今天我又穿上白衬衣和黑裤子，衣服下摆收进裤腰里，我看着电梯镜子里的我，和从前喜欢过的那个男生说，我觉得我看起来像是去推销的。

袖口有些发黄了，也许是在衣柜里放太久了，因为我其实真的没有穿过几次这件衣服。

春天的风很大，吹得我伞都撑不住。我站在操场边看花，舍友都和我说花要谢了，可是我觉得，这是还没完全开呀。春天要来了，春天会来吗？

我勇敢地向HR询问他们在初筛时最看重的是什么，很勇敢地当着在场许多学弟的面说其实我早就网申了但已经收到了感谢信，我很勇敢地在公司站台前坐下，把简历递给HR姐姐。

内向和外向竟然能同时存在于一个人的身上。就像穿上三年前那件白衬衣的我，也忽然像是高三和大三的叠加态。

{% quote %}
往哪个地方拜能请神请到三年前的我呢。

那时候的我也许成绩也并没有达到优秀的标准，但是那个天真的女孩会相信努力可以跨越万难，那个怀揣着对世界最美好愿景的女孩真的会期待春天的降临。
{% endquote %}


## 2026年4月2日 09点06分 

{% quote blue %}
然而、然而
{% endquote %}

然而我们都知道世界并非童话。

投了二十来家公司二十来个岗位，当然比起海投算得上很少的了，有一半早已挂掉，甚至是倒在初筛。拿到两个面试资格，一个是腾讯的群面，面试过程比想象中好，但不出意料地挂了；第二个是另一家大厂的业务面初面，面试过程自我感觉也比较良好，但三天了也没有推进进度也没有约二面，感觉是无了。

某个星期四上软件管理的课，老师是散播焦虑PUA的，点名同学是拷打提问的，气急了要说这样让你们现在的水平拉去考试是开根号乘十都救不了你们的，是要说你们刷题能找到工作哪有怎样呢、几年过后这个岗位可能就没有了。

怎样吗。刚好前几天两个歌手在闹那个事情。我真想回他一句，**如何能又能怎**。

我就这样听着他输出你们要完蛋了的焦虑，一边看到邮箱蹦出来新邮件，_很抱歉经过慎重评估，你暂不匹配……岗位的需求_。

发感谢信的已经算是特意通知你不钓着不养鱼也不泡池子，还好，还有的挂了你也闷声不吭，只在你随机打开某家校招官网查看我的投递时给你开盲盒式的惊喜。流程在HR初筛那一步就终止了，我给同学发消息，我说我想到了一个地狱笑话。

这不就是：

{% quote pink %}
别笑，你也过不了第二关。
{% endquote %}

![2.jpg](https://picui.ogmua.cn/s1/2026/04/02/69cdc75ed865a.webp)

好地狱啊，笑得我有点想哭。

回宿舍的路上我沿着河边走，这条路经过的人实在少，所有的情绪都在初春微凉的风里融化成难以抑制的眼泪。网上说说得了，现实中谁不想急头白脸被老师一通压力过后看到感谢信和流程已中止呢。这简直太轴了，一打一配合地告诉我：你就是很差劲，很烂，你不配。

后来被挂的多了（主要是阿里系，已经在测评阶段连着挂我四次了）我不再会对突然的终止而情绪化。等了很久也终于等到了两个面试，虽然其中一个是腾讯海捞还给我换了个岗位。面完后我不知道为什么感到一阵无来源的委屈，其实没有人故意刁难我，也没有人给我上压力，但我就是委屈。也许把自己的项目经历个人经历剖开来细细地给另一个人讲和裸奔也没什么区别。

那天晚上我不记得我梦到了什么，但总归不是什么愉快的东西。醒来的时候我是平躺着的，可我的左手麻得像是收到了重压。我翻了个身朝右侧躺着，半梦半醒时迷茫地想也许我的手要因为血流不足缺氧截肢了。

**而这仅仅是噩梦的开始**。

我不知道是不是因为焦虑和压力的重重堆叠导致世界终于对我降下了神罚。可是我也无法控制我晚上梦到的东西，好像也没有特别的手段去抵抗睡眠质量的下降。

第二天的晚上，我后半夜几乎是20分钟醒一次的程度。我不知道那天晚上我究竟多少次绝望地从一个又一个破碎的梦中惊醒，又是如何确认自己此时此刻在宿舍而不是家里或着别的地方，重新入睡的。我梦见我拿到了心仪的offer，我梦到我一切努力前功尽弃，我梦到我什么都没有，我梦到……

我不知道我梦到了什么，我觉得那个无人在意也无人知晓的夜晚我真的要崩溃了，没有从十楼一跃而下可能是佛在救我。

<br>

**我觉得我已经不怕面试了。**

**但我开始害怕黑夜和入睡。**
